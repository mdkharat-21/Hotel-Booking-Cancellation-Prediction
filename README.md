# Hotel Booking Cancellation Prediction

## Overview


This project predicts hotel booking cancellations using Data Science techniques & Machine Learning models to identify cancellation patterns and improve prediction accuracy.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## ML Models Used

* Logistic Regression
* KNN
* Decision Tree
* Random Forest

## Workflow

1. Problem Statement
2. Data Collection
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Feature Selection
6. Model Building
7. Hyperparameter Tuning
8. Model Evaluation

## Result

KNN with StandardScaler achieved a training accuracy of 89.55% and testing accuracy of 84.86%.
The small difference between training and testing accuracy indicates that the model generalizes well on unseen data.
Overall, multiple models were compared and performance was evaluated to select the best approach.
